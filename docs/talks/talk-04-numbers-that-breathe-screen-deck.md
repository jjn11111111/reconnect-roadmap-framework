## Production spec

| Setting | Value |
|---------|--------|
| **Canvas** | 1920 × 1080 (16:9) |
| **Safe zone** | 80 px margin — digits never touch edge |
| **Background** | Near-black `#0a0a0f` or deep navy — not pure black (compression banding) |
| **Primary type** | Single weight sans (Inter, Helvetica, SF Pro) — **no thin fonts** |
| **Digit size** | Hero numbers **120–180 pt**; table cells **72 pt minimum** |
| **Accent** | One color only for highlights (e.g. amber `#f5a623` or teal `#2dd4bf`) |
| **Badge** | Top-right pill: `VERIFIED` (green) · `INVITATION` (blue) — see script |
| **Captions** | Burn in OR YouTube auto — assume 40% watch muted first 60 sec |
| **B-roll inserts** | Phone calculator screen capture · optional finger tapping keys |
| **Pattern interrupt** | Hard cut or 0.3 sec zoom every **60–90 sec** in first 5 min |

### Recording order (batch-friendly)

1. Slides 0 + 3–8 as **screen-only** (no VO) — export as B-roll  
2. VO in **5 chunks:** cold open · doubling · six-cycle+seven · pump+primes · 142857+close  
3. Picture-in-picture: calculator cam bottom-right on slides 3, 7, 8 only
