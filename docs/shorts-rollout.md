# Shorts Rollout Plan

**Purpose:** Batch-friendly YouTube Shorts strategy for re:CONNECT — credibility first, funnel to long-form, low speaking load.

**Related:** [youtube-adaptation.md](./youtube-adaptation.md) · [talks/talk-04-numbers-that-breathe-screen-deck.md](./talks/talk-04-numbers-that-breathe-screen-deck.md)

---

## Job of a Short

```
Short = one idea + one visual + one action
Long  = the map
Reel  = proof you can hold a room (3 min / 18 min cuts)
```

Metrics: views · subs from Shorts · **playlist clicks** (did they reach Talk 4?).

---

## Production spec (every Short)

- **9:16** · 1080×1920 · **30–45 sec** (max 60)
- **Hook text frame 1** (mute-safe)
- **No intro** — start mid-calculation or mid-sentence
- **End card 2 sec:** Full talk → playlist
- **Captions** on hook minimum
- Badges: `VERIFIED` · `INVITATION` · `HYPOTHESIS` where applicable

**Batch order:** screen captures → vertical crops → text overlay once (consistent font).
