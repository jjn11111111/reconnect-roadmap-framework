# YouTube & Video Adaptation Guide

**Series:** re:CONNECT — The Power of −0−  
**Purpose:** Translate TED-length talk scripts into a YouTube channel that builds reach *and* a speaking reel — without becoming permanent comment-section defense.  
**North Star:** Same as stage — one felt idea, one verifiable anchor, one invitation. YouTube adds **packaging**, not a second cosmology.

**Related:** [presentation-series.md](./presentation-series.md) · [talks/README.md](./talks/README.md) · [NORTH-STAR.md](../NORTH-STAR.md)

---

## TED room vs YouTube feed

| TED assumption | YouTube reality | Adaptation |
|----------------|-----------------|------------|
| Captive audience, dark room | Swipe in 8 seconds | **Cold open** = best line from minute 12, moved to 0:00 |
| 18–22 min uninterrupted arc | Retention cliffs every 60–90 sec | Pattern interrupt: cut, zoom, on-screen digit, B-roll |
| Verbal disclaimers | Half watch muted | **Badges on screen:** `VERIFIED` · `HYPOTHESIS` · `INVITATION` |
| Live audience exercise | Solo viewer | “Pause here. Try this.” + **30-sec timer** graphic |
| “See you at the next talk” | Algorithm + playlist | End card → **next video in playlist** (auto-play) |

## Production paths (low-speaking load)

Pick **one** primary path; syndicate the export everywhere.

| Path | Speaking load | Best for | Notes |
|------|---------------|----------|-------|
| **A — VO + slides + B-roll** *(recommended)* | Low — record in 2–3 min chunks | Talks 1, 3, 7, 8, 9 | Raindrop, cymatics, calculator, keyboard as hero footage |
| **B — Demo-first / hands-only** | Minimal face | Talks 4, 5, 9 | Screen capture + calculator; face optional 30 sec open/close |
| **C — Essay video** | Chunked VO or collaborator reads | All | Ken Burns style; your scripts already voice well |
| **D — Collaborator host** | You compose; they perform | Boundary series | You on camera only for Talk 6 / monthly Q&A |
| **E — Audio-only syndication** | Zero extra speaking | All | Same export → Spotify/Apple via RSS |

**Batch workflow (injury-friendly):**  
1. Voice-memo cold opens only (one 15-min session).  
2. Transcript → approve → record full VO in **sections**, not one take.  
3. Edit → chapters → upload → mirror to Rumble/podcast.

## Platform roles

| Platform | Role | Priority |
|----------|------|----------|
| **YouTube** | Discovery, Shorts, playlists, **speaking reel host** | Primary |
| **Rumble** | Mirror long-form + boundary talks; less algo anxiety | Syndicate after YT |
| **Spotify / Apple Podcasts** | Audio strip of same export | Zero extra work |
| **Substack / site** | Transcript + diagrams (SEO for “digital root”, “142857”) | Monthly |
| **X / TikTok / Reels** | Vertical crop of Shorts | Repost only |

**Rule:** One edit pipeline. Do not customize for Rumble until **10+ videos** exist on YouTube.

## Universal edit checklist (every long video)

- [ ] **15-sec cold open** before title card (script below per talk)  
- [ ] **Chapters** pasted into description (copy from tables below)  
- [ ] **Pattern interrupt** every 60–90 sec in first 5 min (non-negotiable for retention)  
- [ ] **Hypothesis badge** before any calendar/prison/health/spirit blocks  
- [ ] **Pinned comment:** one verifiable claim + link to framework repo  
- [ ] **End card:** 20 sec — one practice + “Full playlist” (not begging subscribe)  
- [ ] **Thumbnail:** one symbol, 3–5 words, high contrast  
- [ ] **Description line 1:** search-friendly hook (not stage title)

### On-screen badges

```
VERIFIED     — audience can check in 60 sec (Talk 4 math, pH 7, etc.)
HYPOTHESIS   — labeled speculation (Talk 7 spell, 7B suppression, Talk 8 spirit block)
INVITATION   — practice / middle node / pause exercise
```
